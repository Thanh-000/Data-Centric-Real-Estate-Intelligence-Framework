# Legacy Validation Archive

This directory isolates pre-consolidation material that compared multiple DC-REIF tracks before the repository was simplified to one final public-facing system.

Archived here:

- the former Random Forest frozen baseline summaries
- the former candidate v1.6 XGBoost comparison outputs
- old champion-versus-challenger helper scripts

These files are retained only for historical reference. They are not part of the active workflow, the canonical `final_results_master`, or the public notebook/report path.
